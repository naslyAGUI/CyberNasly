Projects.

Senior Project

The Senior project consisted of creating a responsive web application. My group and I decided to make a Meal Plan app that suggested healthy food choices to the user. The user had to create an account, which prompted the user to answer a questionnaire that calculated the user's BMI. Based on the calculated BMI the Meal Plan app recommended healthy meals. Besides recommending healthy meals, the Meal Plan app filtered the user's food search to suitable food choices depending on the user's remaining calories. As part of the team, I was in charge of the Front-end part of the project. I was responsible for the UX design, programming code languages, and had to work with the Back-end developer for deployment. Occasionally, I would help manage the database by creating new database tables from scratch using the terminal. For more detail visit my Github account

Assigment 1 Project

Assignment 1 Project consisted of creating three forms. The first form allowed the user to create an account. The user's password was hashed and was stored with the rest of the information in a database (phpMyAdmin). The second form was a login form that allowed authorized users to perform transactions: withdraw or deposit. The third form asked users for his/her credentials to authenticate the user and display the user's desire number of transactions. The second part of this project (Assignment 2) focused more on securing the user's account by adding an extra layer of security, a captcha.

Ethical Hacking Udemy Course Project

For the Ethical Hacking Udemey Project, I had to set a hacking lab, which is a safe place to experiment and practice hacking. For this, I used VirtualBox, a Kali Linux Machine, a Metasploit machine, a wireless adapter device, and three personal laptops (the local host Windows Machine, a Mac laptop, and another Windows laptop). The first thing I learned to do was to change the MAC address of my wireless adapter. Changing the MAC address increases anonymity helps to impersonate other devices, and bypasses filters. Furthermore, I was able to perform an ARP spoofing attack against two of my laptops using ARP spoofing and Bettercap. I was also able to create a fake access point (evil twin). There were many other things I learned through this course, but I was limited to certain parts mostly due to my wireless adapter device (2.4GHZ frequency). But overall, it was a great way to learn ethical hacking. Remember to use your knowledge for good.

Skills Gained:
System Administration: Kali Linux, Windows 10, macOS
Tools: Bettercap, ARPSpoof, Nmap/zenmap, Wireshark, AirDump-NG, Mana-toolkit
Technologies: Apache2, Sendinblue



WiCyS SANS CTF 2021

Capturing the Flag (CTF) is a cyber security competition where individuals are given tasks such as guessing a password to break into a server or using several security tools to find the hidden message behind a photo. WiCyS SANS CTF was my first CTF. It was a great experience full of learning. I spend three days capturing flags. The challenges included: cryptography, steganography, forensics, network, web, social engineering, and many others.

Skills Gained:

Encoding Schemes: Base64, Hex
Ciphers: Ceasar cipher, Substitution cipher, XOR cipher
Tools: CyberChef, Wireshark, Nmap/Zenmap, nc, 7zip, pcapng analyzer
